🛠️  TROUBLESHOOTING:

   ❌ Backend won't start?
      → Check .env file exists in backend folder
      → Run: pip install -r requirements.txt

   ❌ Frontend won't start?
      → Check Node.js installed
      → Run: npm install

   ❌ Can't connect?
      → Make sure backend is running first
      → Check frontend .env has correct URL
